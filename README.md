# Oscile
Combine music, art, and math! This project was made using the Hack Club Oscillart tutorial and was coded in HTML, CSS, and Javascript. Enter notes in the text box to see its corresponding sine wave and hear how it sounds! Never miss a beat using the built-in ability to record your work. Jam to your favorite song, be a DJ for a day, or make something new to listen to!
